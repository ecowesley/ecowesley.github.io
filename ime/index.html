如何使用你的大新 .CIN？

  1. 從你的電腦選擇 .cin 檔案，點「載入碼表」。本頁會自動解析 %chardef 區塊，建立碼 → 字的索引。
  2. 若你的 .cin 以 U+XXXX 代碼表示字元，也會自動轉碼。
  3. 部分舊檔可能是 Big5 編碼，瀏覽器讀入會亂碼；請先用編輯器另存為 UTF‑8 再載入（如：Notepad++、VS Code）。

備註：請自行確認 .CIN 檔案的授權與使用範圍。官方大新倉頡屬商業軟體，請遵守授權條款。

部署到 GitHub Pages（github.io）

  1. 在 GitHub 建立倉庫（例如 yourname.github.io 或任意專案倉庫）。
  2. 把這個 index.html 檔放進倉庫的根目錄，推送。
  3. 到 Repo → Settings → Pages，Source 選擇 main /root（或 docs/ 資料夾）。
  4. 等綠色勾勾後，開啟顯示的網址，即可在公司電腦上直接使用本頁 IME。
